
 Task 6: K-Nearest Neighbors (KNN) Classification

 Objective
To understand and implement the KNN algorithm for classification problems.

Tools Used
* Python
* Scikit-learn
* NumPy
* Matplotlib

 Dataset
Iris Dataset (built-in from sklearn)

 Steps Performed
1. Loaded the Iris dataset
2. Split data into training and testing sets
3. Normalized the features using StandardScaler
4. Applied KNN algorithm using different values of K
5. Evaluated model using accuracy and confusion matrix
6. Visualized decision boundary using two features

 Results
* Achieved 100% accuracy
* Confusion matrix shows perfect classification
* Decision boundary clearly separates classes

 Conclusion
KNN performs well on structured datasets like Iris. Feature scaling improves performance. Choosing the right K value is important for accuracy.
